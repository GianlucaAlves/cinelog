# Definitions of Done (DoD)

## UI Feature DoD
- Works on mobile + desktop layouts
- Keyboard navigation and visible focus states
- Loading + error state handled
- prefers-reduced-motion supported (motion reduced/disabled)
- No hardcoded secrets
- No console errors

## API Feature DoD
- Validation for inputs
- Proper status codes
- No leaking sensitive fields
- Unit-level checks or at least Postman collection steps documented
